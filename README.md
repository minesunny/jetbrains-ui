## What's included

- **Components**: Essential UI pieces with baseline styles inspired by JetBrains design system, built on [Radix UI](https://www.radix-ui.com/) primitives.
- **Icons**: 1700+ JetBrains IDE icons across 13 categories (general, database, plugins, nodes, vcs, etc.), each with light and dark mode variants and multiple size options. Installable individually or by category via Shadcn CLI.

## Not a library — an open component distribution

Like [shadcn/ui](https://ui.shadcn.com/), JetBrains UI is not a typical install-from-NPM library. It's an open collection you can **copy, modify, and customize** directly in your codebase.
