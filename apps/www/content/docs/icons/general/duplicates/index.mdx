## Overview

A set of **4 Duplicates icons** from the JetBrains UI design system, provided as standalone React components with TypeScript support.

### Usage

<IconDetailPreview name="SendToTheLeft" category="duplicates" />

```tsx
import { SendToTheLeft } from '@/components/jetbrains-ui/icons/general/duplicates';

<SendToTheLeft />

<SendToTheLeft size="lg" mode="dark" />

<SendToTheLeft aria-label="SendToTheLeft icon" />
```
