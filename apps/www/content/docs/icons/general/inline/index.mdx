## Overview

A set of **18 Inline icons** from the JetBrains UI design system, provided as standalone React components with TypeScript support.

### Usage

<IconDetailPreview name="Browse" category="inline" />

```tsx

<Browse />

<Browse size="lg" mode="dark" />

<Browse aria-label="Browse icon" />
```
