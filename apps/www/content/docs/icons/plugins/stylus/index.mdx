## Overview

A set of **1 Stylus icons** from the JetBrains UI design system, provided as standalone React components with TypeScript support.

### Usage

<IconDetailPreview name="Stylus" category="stylus" />

```tsx
import { Stylus } from '@/components/jetbrains-ui/icons/plugins/stylus';

<Stylus />

<Stylus size="lg" mode="dark" />

<Stylus aria-label="Stylus icon" />
```
