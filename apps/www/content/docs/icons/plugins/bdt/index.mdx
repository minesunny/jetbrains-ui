## Overview

A set of **18 BDT icons** from the JetBrains UI design system, provided as standalone React components with TypeScript support.

### Usage

<IconDetailPreview name="ArbitraryCluster" category="bdt" />

```tsx
import { ArbitraryCluster } from '@/components/jetbrains-ui/icons/plugins/bdt';

<ArbitraryCluster />

<ArbitraryCluster size="lg" mode="dark" />

<ArbitraryCluster aria-label="ArbitraryCluster icon" />
```
