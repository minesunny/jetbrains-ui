## Overview

A JetBrains UI icon library provided as standalone React components with TypeScript support. It includes 20 categories such as General, Tool Windows, Actions, and Editor.

### Usage

<IconUsagePreview />

```tsx
import { Add } from '@/components/jetbrains-ui/icons/general/general';

<Add />

<Add size="lg" mode="dark" />

<Add aria-label="Add icon" />
```
