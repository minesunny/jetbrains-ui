## Overview

A JetBrains-style single-line text input component for forms and search fields.

### Design Highlights

- **JetBrains color system**: Built on JetBrains-inspired color tokens.
- **Complete states**: Default, hover, focus, invalid, and disabled states.
- **Compact size**: Optimized for dense settings and form layouts.
- **Accessible by default**: Proper focus ring and keyboard support.

<ComponentInstallation name="input" />

## Usage

```tsx
import { Input } from '@/components/jetbrains-ui/input';

<Input placeholder="Type here..." />;
```

### Invalid State

```tsx
<Input aria-invalid placeholder="Email is required" />
```

### Disabled State

```tsx
<Input disabled placeholder="Disabled input" />
```
