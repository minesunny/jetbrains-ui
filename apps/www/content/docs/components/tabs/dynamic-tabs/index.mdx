## Overview

`Dynamic Tabs` is powered by the `DynamicTabs` component, a wrapper around `Tabs` that renders triggers and contents from an `items` array.

## Usage

### Basic

```tsx
import { DynamicTabs } from '@/components/jetbrains-ui/tabs';

const items = [
  { value: 'overview', label: 'Overview', content: 'Overview content' },
  { value: 'settings', label: 'Settings', content: 'Settings content' },
];

<DynamicTabs items={items} defaultValue="overview" />;
```

### Controlled

```tsx
const [value, setValue] = React.useState('overview');

<DynamicTabs
  items={items}
  value={value}
  onValueChange={setValue}
  orientation="horizontal"
/>;
```
