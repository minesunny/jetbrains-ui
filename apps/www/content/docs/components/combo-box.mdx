## Overview

A combo box combines a drop-down list and an input field, allowing the user to select a value from the list or enter a custom value.
This implementation is built on Radix UI Select with JetBrains UI styling and interaction states.

### Design Highlights

- **JetBrains color system**: Uses colors inspired by the JetBrains UI Kit.
- **Smooth animation**: Soft open/close transition effects.
- **Group support**: Supports grouped options and separators.
- **Accessible by default**: Full keyboard navigation and `focus-visible` styling.

## Usage

```tsx
import {
  ComboBox,
  ComboBoxTrigger,
  ComboBoxValue,
  ComboBoxContent,
  ComboBoxItem,
} from '@/components/jetbrains-ui/combo-box';

<ComboBox>
  <ComboBoxTrigger size="md">
    <ComboBoxValue placeholder="Select an option" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxItem value="option1">Option 1</ComboBoxItem>
    <ComboBoxItem value="option2">Option 2</ComboBoxItem>
    <ComboBoxItem value="option3">Option 3</ComboBoxItem>
  </ComboBoxContent>
</ComboBox>;
```

### With Groups

```tsx
<ComboBox>
  <ComboBoxTrigger size="md">
    <ComboBoxValue placeholder="Select a fruit" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxGroup>
      <ComboBoxLabel>Fruits</ComboBoxLabel>
      <ComboBoxItem value="apple">Apple</ComboBoxItem>
      <ComboBoxItem value="banana">Banana</ComboBoxItem>
    </ComboBoxGroup>
    <ComboBoxSeparator />
    <ComboBoxGroup>
      <ComboBoxLabel>Vegetables</ComboBoxLabel>
      <ComboBoxItem value="carrot">Carrot</ComboBoxItem>
      <ComboBoxItem value="potato">Potato</ComboBoxItem>
    </ComboBoxGroup>
  </ComboBoxContent>
</ComboBox>
```

### Controlled

```tsx
const [value, setValue] = React.useState('');

<ComboBox value={value} onValueChange={setValue}>
  <ComboBoxTrigger size="md">
    <ComboBoxValue placeholder="Select an option" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxItem value="option1">Option 1</ComboBoxItem>
    <ComboBoxItem value="option2">Option 2</ComboBoxItem>
  </ComboBoxContent>
</ComboBox>;
```

### Invalid State

```tsx
<ComboBox>
  <ComboBoxTrigger size="md" aria-invalid={true}>
    <ComboBoxValue placeholder="Select an option" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxItem value="option1">Option 1</ComboBoxItem>
  </ComboBoxContent>
</ComboBox>
```

### Active State

```tsx
<ComboBox>
  <ComboBoxTrigger size="md" active>
    <ComboBoxValue placeholder="Select an option" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxItem value="option1">Option 1</ComboBoxItem>
  </ComboBoxContent>
</ComboBox>
```

### Disabled State

```tsx
<ComboBox disabled>
  <ComboBoxTrigger size="md">
    <ComboBoxValue placeholder="Select an option" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxItem value="option1">Option 1</ComboBoxItem>
  </ComboBoxContent>
</ComboBox>
```

## Sizing Rules

- Trigger size follows Figma default: `224x40`.
- `size="fluid"` and `size="md"` currently share the same visual width (`224px`) for compatibility.
- Dropdown content keeps a minimum width of `224px` to match the trigger.
- Option row height is `28px`.
