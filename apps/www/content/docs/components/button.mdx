## Overview

A JetBrains-style button component built with the Radix UI Slot pattern.

### Design Highlights

- **JetBrains color system**: Uses colors inspired by the JetBrains UI Kit.
- **2 variants**: `primary` and `secondary` for common button scenarios.
- **Figma-aligned sizing**: default `72x28`, slim `72x24`.
- **Optional tooltip**: Built-in tooltip via the existing Tooltip component.
- **Complete states**: Hover, active, disabled, and focus states.
- **Accessible by default**: Includes proper `focus-visible` styling.

## Usage

```tsx
import { Button } from '@/components/jetbrains-ui/buttons';

export default function Demo() {
  return (
    <div className="flex items-center gap-2">
      <Button>Primary</Button>
      <Button variant="secondary">Secondary</Button>
      <Button slim>Slim</Button>
      <Button tooltip="Button tooltip">With tooltip</Button>
    </div>
  );
}
```

### As Child

```tsx
import Link from 'next/link';
import { Button } from '@/components/jetbrains-ui/buttons';

<Button asChild>
  <Link href="/docs/installation">Go to Installation</Link>
</Button>;
```
