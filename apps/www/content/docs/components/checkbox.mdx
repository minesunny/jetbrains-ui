## Overview

Checkbox is used when you need to select multiple items in a list.
Users can select all, any, or none of the items.
This implementation is built on Radix UI Checkbox with JetBrains UI styling.

### Design Highlights

- **Complete state matrix**: Covers checked state and interaction state combinations.
- **JetBrains color system**: Uses colors inspired by the JetBrains UI Kit.
- **Smooth animation**: Elegant check animation with SVG path transitions.
- **Accessible by default**: Full keyboard navigation and `focus-visible` styling.

### State Matrix

| Checked State | Interaction State                                |
| ------------- | ------------------------------------------------ |
| Unchecked     | Default, Hover, Active, Focus, Invalid, Disabled |
| Checked       | Default, Hover, Active, Focus, Invalid, Disabled |
| Indeterminate | Default, Hover, Active, Focus, Invalid, Disabled |

## Sizing Rules

- Checkbox control size is fixed at `16x16`.
- When used with `label`, the field row keeps a minimum height of `32px`.
- Check and indeterminate icons use a `14x14` drawing area.

## Usage

```tsx
<Checkbox defaultChecked />
```

### With Label

```tsx
<Checkbox label="Accept terms and conditions" />
```

### Indeterminate State

```tsx
const [checked, setChecked] = React.useState<boolean | 'indeterminate'>(false);

<Checkbox indeterminate checked={checked} onCheckedChange={setChecked} />;
```

### Invalid State

```tsx
<Checkbox aria-invalid={true} />
```

### Disabled State

```tsx
<Checkbox disabled />
<Checkbox disabled defaultChecked />
```
