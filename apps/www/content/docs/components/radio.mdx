## Overview

A JetBrains-style radio component built on Radix UI RadioGroup, with smooth selection animations.

### Design Highlights

- **Complete state matrix**: Covers checked state and interaction state combinations.
- **JetBrains color system**: Uses colors inspired by the JetBrains UI Kit.
- **Smooth animation**: Elegant selection feedback with spring scale transitions.
- **Accessible by default**: Full keyboard navigation and `focus-visible` styling.

### State Matrix

| Checked State | Interaction State                                |
| ------------- | ------------------------------------------------ |
| Unchecked     | Default, Hover, Active, Focus, Invalid, Disabled |
| Checked       | Default, Hover, Active, Focus, Invalid, Disabled |

## Usage

```tsx
import { RadioGroup, RadioGroupItem } from '@/components/jetbrains-ui/radio';
<RadioGroup defaultValue="option-1">
  <div className="flex items-center gap-2">
    <RadioGroupItem value="option-1" id="option-1" />
    <label htmlFor="option-1">Option 1</label>
  </div>
  <div className="flex items-center gap-2">
    <RadioGroupItem value="option-2" id="option-2" />
    <label htmlFor="option-2">Option 2</label>
  </div>
</RadioGroup>;
```

### Controlled

```tsx
const [value, setValue] = React.useState('option-1');

<RadioGroup value={value} onValueChange={setValue}>
  <RadioGroupItem value="option-1" id="option-1" />
  <RadioGroupItem value="option-2" id="option-2" />
</RadioGroup>;
```

### Invalid State

```tsx
<RadioGroup>
  <RadioGroupItem value="option-1" aria-invalid={true} />
</RadioGroup>
```

### Disabled State

```tsx
<RadioGroup disabled>
  <RadioGroupItem value="option-1" />
  <RadioGroupItem value="option-2" />
</RadioGroup>
```
