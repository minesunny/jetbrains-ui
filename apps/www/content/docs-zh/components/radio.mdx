## 概述

基于 JetBrains UI 设计风格的单选组件，使用 Radix UI RadioGroup 作为基础，提供流畅的选中动画效果。

### 设计特点

- **完整状态矩阵**：支持选中维度与交互维度的完整组合。
- **JetBrains 风格配色**：采用 JetBrains UI Kit 的颜色系统。
- **流畅动画**：弹性缩放动画实现优雅的选中效果。
- **无障碍支持**：完整的键盘导航和 `focus-visible` 状态。

### 状态矩阵

| 选中状态 | 交互状态                           |
| -------- | ---------------------------------- |
| 未选中   | 默认、悬停、按下、聚焦、错误、禁用 |
| 已选中   | 默认、悬停、按下、聚焦、错误、禁用 |

## 用法

```tsx
import { RadioGroup, RadioGroupItem } from '@/components/jetbrains-ui/radio';
<RadioGroup defaultValue="option-1">
  <div className="flex items-center gap-2">
    <RadioGroupItem value="option-1" id="option-1" />
    <label htmlFor="option-1">选项 1</label>
  </div>
  <div className="flex items-center gap-2">
    <RadioGroupItem value="option-2" id="option-2" />
    <label htmlFor="option-2">选项 2</label>
  </div>
</RadioGroup>;
```

### 受控模式

```tsx
const [value, setValue] = React.useState('option-1');

<RadioGroup value={value} onValueChange={setValue}>
  <RadioGroupItem value="option-1" id="option-1" />
  <RadioGroupItem value="option-2" id="option-2" />
</RadioGroup>;
```

### 错误状态

```tsx
<RadioGroup>
  <RadioGroupItem value="option-1" aria-invalid={true} />
</RadioGroup>
```

### 禁用状态

```tsx
<RadioGroup disabled>
  <RadioGroupItem value="option-1" />
  <RadioGroupItem value="option-2" />
</RadioGroup>
```
