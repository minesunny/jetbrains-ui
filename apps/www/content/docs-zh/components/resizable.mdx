## 概述

可组合的可调整面板组件，支持面板折叠与最小尺寸触发滚动兜底。

## 安装

<ComponentInstallation name="resizable" />

## 用法

```tsx
import {
  ResizableHandle,
  ResizablePanel,
  ResizablePanelGroup,
} from '@/components/jetbrains-ui/resizable';

<ResizablePanelGroup direction="horizontal">
  <ResizablePanel defaultSize={40} minSize={20} collapsible collapsedSize={0}>
    A
  </ResizablePanel>
  <ResizableHandle />
  <ResizablePanel defaultSize={60} minSize={30}>
    B
  </ResizablePanel>
</ResizablePanelGroup>;
```

## 用例：可折叠面板 + 最小尺寸滚动兜底

```tsx
'use client';

import * as React from 'react';

import {
  ResizableHandle,
  ResizablePanel,
  ResizablePanelGroup,
} from '@/components/jetbrains-ui/resizable';

export function ResizableCollapsibleExample() {
  const [collapse, setCollapse] = React.useState(false);

  return (
    <div className="space-y-3">
      <button
        type="button"
        className="rounded border px-3 py-1 text-sm"
        onClick={() => setCollapse((prev) => !prev)}
      >
        切换 Panel A
      </button>

      <ResizablePanelGroup
        direction="horizontal"
        className="h-56 rounded-lg border"
      >
        <ResizablePanel
          defaultSize={45}
          minSize={20}
          collapsible
          collapse={collapse}
          collapsedSize={0}
        >
          <div className="h-full min-w-[640px] p-3">
            当面板缩到 minSize 时，内容会切换为面板内部滚动。
          </div>
        </ResizablePanel>
        <ResizableHandle withHandle />
        <ResizablePanel defaultSize={55} minSize={25}>
          <div className="h-full p-3">Panel B</div>
        </ResizablePanel>
      </ResizablePanelGroup>
    </div>
  );
}
```
