## 概述

可组合的可调整面板组件，支持面板折叠与尺寸记忆。

`collapsedSize` 默认值为 `0`；设为 `-1` 表示禁用折叠。`collapsed` 用于控制当前折叠状态。

## 安装

<ComponentInstallation name="resizable" />

## 用法

```tsx
import {
  ResizableHandle,
  ResizablePanel,
  ResizablePanelGroup,
} from '@/components/jetbrains-ui/resizable';

<ResizablePanelGroup direction="horizontal">
  <ResizablePanel defaultSize={40} minSize={20} collapsedSize={0}>
    A
  </ResizablePanel>
  <ResizableHandle />
  <ResizablePanel defaultSize={60} minSize={30}>
    B
  </ResizablePanel>
</ResizablePanelGroup>;
```

## 用例：可折叠面板

```tsx
'use client';

import * as React from 'react';

import {
  ResizableHandle,
  ResizablePanel,
  ResizablePanelGroup,
} from '@/components/jetbrains-ui/resizable';

export function ResizableCollapsibleExample() {
  const [collapsed, setCollapsed] = React.useState(false);

  return (
    <div className="space-y-3">
      <button
        type="button"
        className="rounded border px-3 py-1 text-sm"
        onClick={() => setCollapsed((prev) => !prev)}
      >
        切换 Panel A
      </button>

      <ResizablePanelGroup
        direction="horizontal"
        className="h-56 rounded-lg border"
      >
        <ResizablePanel
          defaultSize={45}
          minSize={20}
          collapsedSize={0}
          collapsed={collapsed}
          onCollapse={() => setCollapsed(true)}
          onExpand={() => setCollapsed(false)}
        >
          <div className="h-full p-3">Panel A</div>
        </ResizablePanel>
        <ResizableHandle withHandle />
        <ResizablePanel defaultSize={55} minSize={25}>
          <div className="h-full p-3">Panel B</div>
        </ResizablePanel>
      </ResizablePanelGroup>
    </div>
  );
}
```
