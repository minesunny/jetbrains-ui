## 概述

组合框结合了下拉列表与输入框，用户既可以从列表中选择值，也可以输入自定义值。
本实现基于 Radix UI Select，并对齐 JetBrains UI 风格与交互状态。

### 设计特点

- **JetBrains 风格配色**：采用 JetBrains UI Kit 的颜色系统。
- **流畅动画**：平滑的展开与收起过渡效果。
- **分组支持**：支持选项分组与分隔线。
- **无障碍支持**：完整的键盘导航和 `focus-visible` 状态。

## 用法

```tsx
import {
  ComboBox,
  ComboBoxTrigger,
  ComboBoxValue,
  ComboBoxContent,
  ComboBoxItem,
} from '@/components/jetbrains-ui/combo-box';

<ComboBox>
  <ComboBoxTrigger size="md">
    <ComboBoxValue placeholder="请选择一个选项" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxItem value="option1">选项 1</ComboBoxItem>
    <ComboBoxItem value="option2">选项 2</ComboBoxItem>
    <ComboBoxItem value="option3">选项 3</ComboBoxItem>
  </ComboBoxContent>
</ComboBox>;
```

### 分组用法

```tsx
<ComboBox>
  <ComboBoxTrigger size="md">
    <ComboBoxValue placeholder="请选择一个水果" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxGroup>
      <ComboBoxLabel>水果</ComboBoxLabel>
      <ComboBoxItem value="apple">苹果</ComboBoxItem>
      <ComboBoxItem value="banana">香蕉</ComboBoxItem>
    </ComboBoxGroup>
    <ComboBoxSeparator />
    <ComboBoxGroup>
      <ComboBoxLabel>蔬菜</ComboBoxLabel>
      <ComboBoxItem value="carrot">胡萝卜</ComboBoxItem>
      <ComboBoxItem value="potato">土豆</ComboBoxItem>
    </ComboBoxGroup>
  </ComboBoxContent>
</ComboBox>
```

### 受控模式

```tsx
const [value, setValue] = React.useState('');

<ComboBox value={value} onValueChange={setValue}>
  <ComboBoxTrigger size="md">
    <ComboBoxValue placeholder="请选择一个选项" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxItem value="option1">选项 1</ComboBoxItem>
    <ComboBoxItem value="option2">选项 2</ComboBoxItem>
  </ComboBoxContent>
</ComboBox>;
```

### 错误状态

```tsx
<ComboBox>
  <ComboBoxTrigger size="md" aria-invalid={true}>
    <ComboBoxValue placeholder="请选择一个选项" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxItem value="option1">选项 1</ComboBoxItem>
  </ComboBoxContent>
</ComboBox>
```

### Active 状态

```tsx
<ComboBox>
  <ComboBoxTrigger size="md" active>
    <ComboBoxValue placeholder="请选择一个选项" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxItem value="option1">选项 1</ComboBoxItem>
  </ComboBoxContent>
</ComboBox>
```

### 禁用状态

```tsx
<ComboBox disabled>
  <ComboBoxTrigger size="md">
    <ComboBoxValue placeholder="请选择一个选项" />
  </ComboBoxTrigger>
  <ComboBoxContent>
    <ComboBoxItem value="option1">选项 1</ComboBoxItem>
  </ComboBoxContent>
</ComboBox>
```

## 尺寸规则

- Trigger 尺寸遵循 Figma 默认值：`224x40`。
- `size="fluid"` 与 `size="md"` 当前保持相同视觉宽度（`224px`），用于兼容现有调用。
- 下拉内容最小宽度为 `224px`，与 Trigger 对齐。
- 选项行高度为 `28px`。
