## 概述

Checkbox 用于在列表中选择多个项目。
用户可以选择全部、部分或不选择任何项目。
本实现基于 Radix UI Checkbox，并对齐 JetBrains UI 风格。

### 设计特点

- **完整状态矩阵**：支持选中维度与交互维度的完整组合。
- **JetBrains 风格配色**：采用 JetBrains UI Kit 的颜色系统。
- **流畅动画**：SVG 路径动画实现优雅的勾选效果。
- **无障碍支持**：完整的键盘导航和 `focus-visible` 状态。

### 状态矩阵

| 选中状态 | 交互状态                           |
| -------- | ---------------------------------- |
| 未选中   | 默认、悬停、按下、聚焦、错误、禁用 |
| 已选中   | 默认、悬停、按下、聚焦、错误、禁用 |
| 半选中   | 默认、悬停、按下、聚焦、错误、禁用 |

## 尺寸规则

- Checkbox 控件尺寸固定为 `16x16`。
- 配合 `label` 使用时，整行最小高度为 `32px`。
- 勾选与半选图标使用 `14x14` 绘制区域。

## 用法

```tsx
<Checkbox defaultChecked />
```

### 搭配标签

```tsx
<Checkbox label="同意条款与条件" />
```

### 半选状态

```tsx
const [checked, setChecked] = React.useState<boolean | 'indeterminate'>(false);

<Checkbox indeterminate checked={checked} onCheckedChange={setChecked} />;
```

### 错误状态

```tsx
<Checkbox aria-invalid={true} />
```

### 禁用状态

```tsx
<Checkbox disabled />
<Checkbox disabled defaultChecked />
```
