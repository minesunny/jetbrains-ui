## 概述

基于 JetBrains UI 设计风格的下拉菜单组件。

## 安装

<ComponentInstallation name="dropdown-menu" />

## 用法

```tsx
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuTrigger,
} from '@/components/jetbrains-ui/dropdown-menu';
import { Button } from '@/components/jetbrains-ui/buttons';

<DropdownMenu>
  <DropdownMenuTrigger asChild>
    <Button>Open Menu</Button>
  </DropdownMenuTrigger>
  <DropdownMenuContent>
    <DropdownMenuItem>Profile</DropdownMenuItem>
  </DropdownMenuContent>
</DropdownMenu>;
```
