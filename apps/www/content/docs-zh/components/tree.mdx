## 概述

用于层级数据展示的 JetBrains 风格树形组件。

## 安装

<ComponentInstallation name="tree" />

## 用法

```tsx
import { Tree, type TreeItemData } from '@/components/jetbrains-ui/tree';

const items: TreeItemData[] = [
  {
    id: 'src',
    label: 'src',
    children: [{ id: 'src-index', label: 'index.tsx' }],
  },
];

<Tree items={items} defaultExpandedIds={['src']} />;
```
