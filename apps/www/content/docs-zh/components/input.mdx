## 概述

JetBrains 风格的单行文本输入组件，适用于表单与搜索场景。

### 设计特点

- **JetBrains 风格配色**：基于 JetBrains 设计令牌实现。
- **完整状态支持**：default、hover、focus、invalid、disabled。
- **紧凑尺寸**：适合高密度设置页与表单布局。
- **无障碍支持**：默认提供清晰的焦点样式与键盘可访问性。

## 用法

```tsx
import { Input } from '@/components/jetbrains-ui/input';

<Input placeholder="请输入内容..." />;
```

### 无效状态

```tsx
<Input aria-invalid placeholder="邮箱为必填项" />
```

### 禁用状态

```tsx
<Input disabled placeholder="禁用输入框" />
```
