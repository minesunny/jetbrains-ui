## 概述

`Tabs` 提供标准组合式 API：

- `Tabs`
- `TabsList`
- `TabsTrigger`
- `TabsContent`

## 用法

### 基本用法

```tsx
import {
  Tabs,
  TabsList,
  TabsTrigger,
  TabsContent,
} from '@/components/jetbrains-ui/tabs';

<Tabs defaultValue="overview">
  <TabsList>
    <TabsTrigger value="overview">概览</TabsTrigger>
    <TabsTrigger value="settings">设置</TabsTrigger>
  </TabsList>
  <TabsContent value="overview">概览内容</TabsContent>
  <TabsContent value="settings">设置内容</TabsContent>
</Tabs>;
```

### 垂直布局

```tsx
<Tabs defaultValue="overview" orientation="vertical" className="flex gap-3">
  <TabsList className="h-auto flex-col items-stretch">
    <TabsTrigger value="overview">概览</TabsTrigger>
    <TabsTrigger value="settings">设置</TabsTrigger>
  </TabsList>
  <TabsContent value="overview">概览内容</TabsContent>
  <TabsContent value="settings">设置内容</TabsContent>
</Tabs>
```
