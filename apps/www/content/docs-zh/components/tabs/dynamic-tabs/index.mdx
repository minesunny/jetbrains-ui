## 概述

`Dynamic Tabs` 由 `DynamicTabs` 组件实现，是对 `Tabs` 的封装，可通过 `items` 数组自动渲染触发器和内容区。

## 用法

### 基本用法

```tsx
import { DynamicTabs } from '@/components/jetbrains-ui/tabs';

const items = [
  { value: 'overview', label: '概览', content: '概览内容' },
  { value: 'settings', label: '设置', content: '设置内容' },
];

<DynamicTabs items={items} defaultValue="overview" />;
```

### 受控模式

```tsx
const [value, setValue] = React.useState('overview');

<DynamicTabs
  items={items}
  value={value}
  onValueChange={setValue}
  orientation="horizontal"
/>;
```
