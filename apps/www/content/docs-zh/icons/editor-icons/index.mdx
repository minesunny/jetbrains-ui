## Overview

来自 JetBrains UI 设计系统的 **67 个 Editor Icons 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="InlayGlobe" category="editor-icons" />

```tsx
import { InlayGlobe } from '@/components/jetbrains-ui/icons/editor-icons';

<InlayGlobe />

<InlayGlobe size="lg" mode="dark" />

<InlayGlobe aria-label="InlayGlobe icon" />
```
