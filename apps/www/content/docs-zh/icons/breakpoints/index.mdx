## Overview

来自 JetBrains UI 设计系统的 **39 个 Breakpoints 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Breakpoint" category="breakpoints" />

```tsx
import { Breakpoint } from '@/components/jetbrains-ui/icons/breakpoints';

<Breakpoint />

<Breakpoint size="lg" mode="dark" />

<Breakpoint aria-label="Breakpoint icon" />
```
