## Overview

来自 JetBrains UI 设计系统的 **18 个 Debugger 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AddToWatch" category="debugger" />

```tsx
import { AddToWatch } from '@/components/jetbrains-ui/icons/debugger';

<AddToWatch />

<AddToWatch size="lg" mode="dark" />

<AddToWatch aria-label="AddToWatch icon" />
```
