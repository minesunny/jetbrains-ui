## Overview

来自 JetBrains UI 设计系统的 **4 个 Terminal 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Command" category="terminal" />

```tsx
import { Command } from '@/components/jetbrains-ui/icons/terminal';

<Command />

<Command size="lg" mode="dark" />

<Command aria-label="Command icon" />
```
