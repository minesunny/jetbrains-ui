## Overview

来自 JetBrains UI 设计系统的 **1 个 Dev Containers 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="DevContainers" category="dev-containers" />

```tsx
import { DevContainers } from '@/components/jetbrains-ui/icons/plugins/dev-containers';

<DevContainers />

<DevContainers size="lg" mode="dark" />

<DevContainers aria-label="DevContainers icon" />
```
