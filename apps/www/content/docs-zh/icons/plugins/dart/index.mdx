## Overview

来自 JetBrains UI 设计系统的 **6 个 Dart 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Dart" category="dart" />

```tsx
import { Dart } from '@/components/jetbrains-ui/icons/plugins/dart';

<Dart />

<Dart size="lg" mode="dark" />

<Dart aria-label="Dart icon" />
```
