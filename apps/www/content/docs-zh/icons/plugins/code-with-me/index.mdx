## Overview

来自 JetBrains UI 设计系统的 **25 个 Code With Me 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="CwmCamAvatarOff" category="code-with-me" />

```tsx
import { CwmCamAvatarOff } from '@/components/jetbrains-ui/icons/plugins/code-with-me';

<CwmCamAvatarOff />

<CwmCamAvatarOff size="lg" mode="dark" />

<CwmCamAvatarOff aria-label="CwmCamAvatarOff icon" />
```
