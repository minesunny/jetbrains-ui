## Overview

来自 JetBrains UI 设计系统的 **2 个 Tasks 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="StartTimer" category="tasks" />

```tsx
import { StartTimer } from '@/components/jetbrains-ui/icons/plugins/tasks';

<StartTimer />

<StartTimer size="lg" mode="dark" />

<StartTimer aria-label="StartTimer icon" />
```
