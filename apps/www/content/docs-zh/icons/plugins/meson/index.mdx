## Overview

来自 JetBrains UI 设计系统的 **4 个 Meson 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Meson" category="meson" />

```tsx
import { Meson } from '@/components/jetbrains-ui/icons/plugins/meson';

<Meson />

<Meson size="lg" mode="dark" />

<Meson aria-label="Meson icon" />
```
