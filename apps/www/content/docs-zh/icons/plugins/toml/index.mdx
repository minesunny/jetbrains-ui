## Overview

来自 JetBrains UI 设计系统的 **1 个 TOML 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Toml" category="toml" />

```tsx
import { Toml } from '@/components/jetbrains-ui/icons/plugins/toml';

<Toml />

<Toml size="lg" mode="dark" />

<Toml aria-label="Toml icon" />
```
