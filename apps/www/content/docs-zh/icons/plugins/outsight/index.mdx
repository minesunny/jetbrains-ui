## Overview

来自 JetBrains UI 设计系统的 **2 个 OutSight 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="OutSight" category="outsight" />

```tsx
import { OutSight } from '@/components/jetbrains-ui/icons/plugins/outsight';

<OutSight />

<OutSight size="lg" mode="dark" />

<OutSight aria-label="OutSight icon" />
```
