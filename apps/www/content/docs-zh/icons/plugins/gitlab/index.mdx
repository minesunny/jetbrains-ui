## Overview

来自 JetBrains UI 设计系统的 **2 个 GitLab 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="GitLabLogo" category="gitlab" />

```tsx
import { GitLabLogo } from '@/components/jetbrains-ui/icons/plugins/gitlab';

<GitLabLogo />

<GitLabLogo size="lg" mode="dark" />

<GitLabLogo aria-label="GitLabLogo icon" />
```
