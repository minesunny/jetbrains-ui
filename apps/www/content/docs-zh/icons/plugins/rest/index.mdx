## Overview

来自 JetBrains UI 设计系统的 **1 个 REST 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Http_request" category="rest" />

```tsx
import { Http_request } from '@/components/jetbrains-ui/icons/plugins/rest';

<Http_request />

<Http_request size="lg" mode="dark" />

<Http_request aria-label="Http_request icon" />
```
