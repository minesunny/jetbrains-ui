## Overview

来自 JetBrains UI 设计系统的 **92 个 JPA 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AddChangelog" category="jpa" />

```tsx
import { AddChangelog } from '@/components/jetbrains-ui/icons/plugins/jpa';

<AddChangelog />

<AddChangelog size="lg" mode="dark" />

<AddChangelog aria-label="AddChangelog icon" />
```
