## Overview

来自 JetBrains UI 设计系统的 **7 个 Bazel 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Bazel" category="bazel" />

```tsx
import { Bazel } from '@/components/jetbrains-ui/icons/plugins/bazel';

<Bazel />

<Bazel size="lg" mode="dark" />

<Bazel aria-label="Bazel icon" />
```
