## Overview

来自 JetBrains UI 设计系统的 **91 个 Java EE 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AbstractClass" category="java-ee" />

```tsx
import { AbstractClass } from '@/components/jetbrains-ui/icons/plugins/java-ee';

<AbstractClass />

<AbstractClass size="lg" mode="dark" />

<AbstractClass aria-label="AbstractClass icon" />
```
