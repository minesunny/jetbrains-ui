## Overview

来自 JetBrains UI 设计系统的 **15 个 Package Checker 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="HighAll" category="package-checker" />

```tsx
import { HighAll } from '@/components/jetbrains-ui/icons/plugins/package-checker';

<HighAll />

<HighAll size="lg" mode="dark" />

<HighAll aria-label="HighAll icon" />
```
