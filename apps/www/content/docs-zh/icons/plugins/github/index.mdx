## Overview

来自 JetBrains UI 设计系统的 **2 个 GitHub 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Github" category="github" />

```tsx
import { Github } from '@/components/jetbrains-ui/icons/plugins/github';

<Github />

<Github size="lg" mode="dark" />

<Github aria-label="Github icon" />
```
