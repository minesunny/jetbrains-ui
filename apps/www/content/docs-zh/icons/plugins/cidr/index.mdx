## Overview

来自 JetBrains UI 设计系统的 **112 个 CIDR 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="CMake" category="cidr" />

```tsx
import { CMake } from '@/components/jetbrains-ui/icons/plugins/cidr';

<CMake />

<CMake size="lg" mode="dark" />

<CMake aria-label="CMake icon" />
```
