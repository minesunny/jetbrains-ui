## Overview

来自 JetBrains UI 设计系统的 **3 个 Velocity 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Velocity" category="velocity" />

```tsx
import { Velocity } from '@/components/jetbrains-ui/icons/plugins/velocity';

<Velocity />

<Velocity size="lg" mode="dark" />

<Velocity aria-label="Velocity icon" />
```
