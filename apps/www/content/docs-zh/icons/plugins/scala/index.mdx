## Overview

来自 JetBrains UI 设计系统的 **55 个 Scala 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AbstractClassObjectScala" category="scala" />

```tsx
import { AbstractClassObjectScala } from '@/components/jetbrains-ui/icons/plugins/scala';

<AbstractClassObjectScala />

<AbstractClassObjectScala size="lg" mode="dark" />

<AbstractClassObjectScala aria-label="AbstractClassObjectScala icon" />
```
