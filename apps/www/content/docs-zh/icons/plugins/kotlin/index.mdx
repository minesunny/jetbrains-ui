## Overview

来自 JetBrains UI 设计系统的 **36 个 Kotlin 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AbstractClassKotlin" category="kotlin" />

```tsx
import { AbstractClassKotlin } from '@/components/jetbrains-ui/icons/plugins/kotlin';

<AbstractClassKotlin />

<AbstractClassKotlin size="lg" mode="dark" />

<AbstractClassKotlin aria-label="AbstractClassKotlin icon" />
```
