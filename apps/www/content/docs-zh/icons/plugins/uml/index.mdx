## Overview

来自 JetBrains UI 设计系统的 **5 个 UML 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AddNote" category="uml" />

```tsx
import { AddNote } from '@/components/jetbrains-ui/icons/plugins/uml';

<AddNote />

<AddNote size="lg" mode="dark" />

<AddNote aria-label="AddNote icon" />
```
