## Overview

来自 JetBrains UI 设计系统的 **6 个 PlatformIO 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Clean" category="platformio" />

```tsx
import { Clean } from '@/components/jetbrains-ui/icons/plugins/platformio';

<Clean />

<Clean size="lg" mode="dark" />

<Clean aria-label="Clean icon" />
```
