## Overview

来自 JetBrains UI 设计系统的 **7 个 Docker 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Docker" category="docker" />

```tsx
import { Docker } from '@/components/jetbrains-ui/icons/plugins/docker';

<Docker />

<Docker size="lg" mode="dark" />

<Docker aria-label="Docker icon" />
```
