## Overview

来自 JetBrains UI 设计系统的 **9 个 AOP 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AbstractAdvice" category="aop" />

```tsx
import { AbstractAdvice } from '@/components/jetbrains-ui/icons/plugins/aop';

<AbstractAdvice />

<AbstractAdvice size="lg" mode="dark" />

<AbstractAdvice aria-label="AbstractAdvice icon" />
```
