## Overview

来自 JetBrains UI 设计系统的 **2 个 Full Line Completion 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="FullLineCompletion" category="full-line-completion" />

```tsx
import { FullLineCompletion } from '@/components/jetbrains-ui/icons/plugins/full-line-completion';

<FullLineCompletion />

<FullLineCompletion size="lg" mode="dark" />

<FullLineCompletion aria-label="FullLineCompletion icon" />
```
