## Overview

来自 JetBrains UI 设计系统的 **1 个 Databricks 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="RunAsWorkflow" category="databricks" />

```tsx
import { RunAsWorkflow } from '@/components/jetbrains-ui/icons/plugins/databricks';

<RunAsWorkflow />

<RunAsWorkflow size="lg" mode="dark" />

<RunAsWorkflow aria-label="RunAsWorkflow icon" />
```
