## Overview

来自 JetBrains UI 设计系统的 **46 个 VCS 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AddEmoji" category="vcs" />

```tsx
import { AddEmoji } from '@/components/jetbrains-ui/icons/vcs';

<AddEmoji />

<AddEmoji size="lg" mode="dark" />

<AddEmoji aria-label="AddEmoji icon" />
```
