## Overview

来自 JetBrains UI 设计系统的 **46 个 Run 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="FailedInProgress" category="run" />

```tsx
import { FailedInProgress } from '@/components/jetbrains-ui/icons/run';

<FailedInProgress />

<FailedInProgress size="lg" mode="dark" />

<FailedInProgress aria-label="FailedInProgress icon" />
```
