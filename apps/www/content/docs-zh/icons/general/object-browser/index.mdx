## Overview

来自 JetBrains UI 设计系统的 **10 个 Object Browser 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AbbreviatePackageNames" category="object-browser" />

```tsx
import { AbbreviatePackageNames } from '@/components/jetbrains-ui/icons/general/object-browser';

<AbbreviatePackageNames />

<AbbreviatePackageNames size="lg" mode="dark" />

<AbbreviatePackageNames aria-label="AbbreviatePackageNames icon" />
```
