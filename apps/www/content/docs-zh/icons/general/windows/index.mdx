## Overview

来自 JetBrains UI 设计系统的 **16 个 Windows 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Close" category="windows" />

```tsx
import { Close } from '@/components/jetbrains-ui/icons/general/windows';

<Close />

<Close size="lg" mode="dark" />

<Close aria-label="Close icon" />
```
