## Overview

来自 JetBrains UI 设计系统的 **121 个 Tool Windows 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Terminal" category="tool-windows" />

```tsx
import { Terminal } from '@/components/jetbrains-ui/icons/general/tool-windows';

<Terminal />

<Terminal size="lg" mode="dark" />

<Terminal aria-label="Terminal icon" />
```
