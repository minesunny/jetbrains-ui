## Overview

来自 JetBrains UI 设计系统的 **5 个 Setting Sync 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Conflict" category="setting-sync" />

```tsx
import { Conflict } from '@/components/jetbrains-ui/icons/general/setting-sync';

<Conflict />

<Conflict size="lg" mode="dark" />

<Conflict aria-label="Conflict icon" />
```
