## Overview

来自 JetBrains UI 设计系统的 **3 个 DevKit 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AddSDK" category="devkit" />

```tsx
import { AddSDK } from '@/components/jetbrains-ui/icons/general/devkit';

<AddSDK />

<AddSDK size="lg" mode="dark" />

<AddSDK aria-label="AddSDK icon" />
```
