## Overview

来自 JetBrains UI 设计系统的 **1 个 Bookmarks 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Bookmarks" category="bookmarks" />

```tsx
import { Bookmarks } from '@/components/jetbrains-ui/icons/general/bookmarks';

<Bookmarks />

<Bookmarks size="lg" mode="dark" />

<Bookmarks aria-label="Bookmarks icon" />
```
