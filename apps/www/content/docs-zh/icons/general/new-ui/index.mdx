## Overview

来自 JetBrains UI 设计系统的 **6 个 New UI 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="DarkTheme" category="new-ui" />

```tsx
import { DarkTheme } from '@/components/jetbrains-ui/icons/general/new-ui';

<DarkTheme />

<DarkTheme size="lg" mode="dark" />

<DarkTheme aria-label="DarkTheme icon" />
```
