## Overview

来自 JetBrains UI 设计系统的 **4 个 Rating 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="FeedbackRating" category="rating" />

```tsx
import { FeedbackRating } from '@/components/jetbrains-ui/icons/general/rating';

<FeedbackRating />

<FeedbackRating size="lg" mode="dark" />

<FeedbackRating aria-label="FeedbackRating icon" />
```
