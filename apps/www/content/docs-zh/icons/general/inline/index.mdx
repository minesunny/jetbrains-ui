## Overview

来自 JetBrains UI 设计系统的 **18 个 Inline 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Browse" category="inline" />

```tsx
import { Browse } from '@/components/jetbrains-ui/icons/general/inline';

<Browse />

<Browse size="lg" mode="dark" />

<Browse aria-label="Browse icon" />
```
