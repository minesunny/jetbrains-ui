## Overview

来自 JetBrains UI 设计系统的 **6 个 Progress Bar 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Pause" category="progress-bar" />

```tsx
import { Pause } from '@/components/jetbrains-ui/icons/general/progress-bar';

<Pause />

<Pause size="lg" mode="dark" />

<Pause aria-label="Pause icon" />
```
