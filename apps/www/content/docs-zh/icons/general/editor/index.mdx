## Overview

来自 JetBrains UI 设计系统的 **18 个 Editor 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AddColumnLeft" category="editor" />

```tsx
import { AddColumnLeft } from '@/components/jetbrains-ui/icons/general/editor';

<AddColumnLeft />

<AddColumnLeft size="lg" mode="dark" />

<AddColumnLeft aria-label="AddColumnLeft icon" />
```
