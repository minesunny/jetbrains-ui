## Overview

来自 JetBrains UI 设计系统的 **13 个 Status 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Error" category="status" />

```tsx
import { Error } from '@/components/jetbrains-ui/icons/general/status';

<Error />

<Error size="lg" mode="dark" />

<Error aria-label="Error icon" />
```
