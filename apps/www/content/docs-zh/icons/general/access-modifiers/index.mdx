## Overview

来自 JetBrains UI 设计系统的 **4 个 Access Modifiers 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="AccessLocal" category="access-modifiers" />

```tsx
import { AccessLocal } from '@/components/jetbrains-ui/icons/general/access-modifiers';

<AccessLocal />

<AccessLocal size="lg" mode="dark" />

<AccessLocal aria-label="AccessLocal icon" />
```
