## Overview

来自 JetBrains UI 设计系统的 **4 个 Usages 图标**，以独立 React 组件形式提供，支持 TypeScript。

### Usage

<IconDetailPreview name="Filterdups" category="usages" />

```tsx
import { Filterdups } from '@/components/jetbrains-ui/icons/general/usages';

<Filterdups />

<Filterdups size="lg" mode="dark" />

<Filterdups aria-label="Filterdups icon" />
```
