## Overview

来自 JetBrains UI 设计系统的图标库，以独立 React 组件形式提供，支持 TypeScript。包含 20 个分类，涵盖 General、Tool Windows、Actions、Editor 等。

### Features

- **5 sizes**: `xs` (12px), `sm` (14px), `md` (16px), `lg` (20px), `xl` (24px)
- **Light & Dark** theme variants
- **Accessibility**: `aria-label` and `title` props with proper `role` attributes
- **Tree-shakeable**: Import only what you need

## Usage

<IconUsagePreview />

```tsx
import { Search } from '@/components/jetbrains-ui/icons/general/general';
import { Terminal } from '@/components/jetbrains-ui/icons/general/tool-windows';

// Basic
<Search />

// With size and theme
<Terminal size="lg" mode="dark" />

// With accessibility
<Search size="md" aria-label="Search icon" title="Search" />
```
