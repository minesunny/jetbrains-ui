**JetBrains UI 是一个基于 React 的组件分发仓库**，构建于 [Tailwind CSS](https://tailwindcss.com/)、[Radix UI](https://www.radix-ui.com/) 与 [shadcn/ui](https://ui.shadcn.com/) 之上。我们的目标是提供高质量、可访问的 UI 组件，以及 1700+ JetBrains IDE 图标，方便你快速集成到项目中。

## 不是传统组件库，而是开放式组件分发

和 [shadcn/ui](https://ui.shadcn.com/) 一样，**JetBrains UI 不是“从 NPM 安装后直接调用”的黑盒组件库**。它是一套开放源码集合，你可以直接 **复制、修改并按需定制** 到自己的代码库中。这样的模式可以避免封装层开销，也不会被样式实现细节限制。

## 包含内容

1. **组件（Components）**：基于 [Radix UI](https://www.radix-ui.com/) 原语构建，提供 JetBrains 风格的基础视觉与交互。
2. **图标（Icons）**：1700+ JetBrains IDE 图标，覆盖 13 个分类（general、database、plugins、nodes 等），同时提供明暗主题与多尺寸能力。

## 为什么选择 JetBrains UI？

1. **开源 + Copy-first**：可直接访问源码，深度适配你的设计系统。
2. **1700+ JetBrains 图标**：完整图标集，支持明暗主题，可通过 Shadcn CLI 按分类或按单个图标安装。
3. **性能与可访问性**：默认配置注重性能与易用性。
4. **构建于 Radix UI**：以可访问、无样式原语为基础。
5. **可与 shadcn/ui 互补使用**：在现有 shadcn/ui 项目中无缝增加 JetBrains 风格组件。
